---
title: 'Sign Up'
api: 'POST /v1/auth/signup'
description: 'Register a new developer account on Avenue.'
---

## Request Body

<ParamField body="email" type="string" required>
  The email address of the developer.
</ParamField>
<ParamField body="password" type="string" required>
  A strong password (min 8 characters).
</ParamField>
<ParamField body="company_name" type="string" required>
  The name of the company or platform.
</ParamField>

## Response

<ResponseExample>
```json
{
  "success": true,
  "data": {
    "status": "success",
    "message": "Account created. Please check your email for the verification link."
  }
}
```
</ResponseExample>
