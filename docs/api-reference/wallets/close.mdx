---
title: 'Close Wallet'
api: 'POST /v1/wallets/:wallet_id/close'
description: 'Deactivate a wallet. Any future inbound transfers to its NUBAN will be routed to the Suspense Queue.'
---

## Request Headers

<ParamField header="x-api-key" type="string" required>
  Your live or test API key.
</ParamField>

## Path Parameters

<ParamField path="wallet_id" type="string" required>
  The Avenue wallet ID.
</ParamField>

## Response

<ResponseExample>
```json
{
  "wallet_id": "wal_abc123",
  "status": "CLOSED",
  "message": "Wallet has been successfully closed."
}
```
</ResponseExample>
