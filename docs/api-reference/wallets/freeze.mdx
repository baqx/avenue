---
title: 'Freeze Wallet'
api: 'POST /v1/wallets/:wallet_id/freeze'
description: 'Temporarily freeze a wallet. Incoming payments will go to Suspense until unfrozen.'
---

## Request Headers

<ParamField header="x-api-key" type="string" required>
  Your live or test API key.
</ParamField>

## Path Parameters

<ParamField path="wallet_id" type="string" required>
  The Avenue wallet ID.
</ParamField>

## Response

<ResponseExample>
```json
{
  "wallet_id": "wal_abc123",
  "status": "FROZEN",
  "message": "Wallet has been successfully frozen."
}
```
</ResponseExample>
