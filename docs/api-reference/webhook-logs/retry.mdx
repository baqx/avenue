---
title: 'Retry Webhook'
api: 'POST /v1/webhook-logs/:log_id/retry'
description: 'Manually trigger a retry for a failed outbound webhook delivery.'
---

<ParamField header="x-api-key" type="string" required>

</ParamField>

## Path Parameters

</ParamField>
